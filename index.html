Minecraft Mod

BitO'Everything is a Java Minecraft mod currently being developed by me and another student. In this mod, a few simple features have been developed, such as new wood types, blocks, and simple items. There are many advanced features though:

Magic - requires firing an event within an interface, so the children receive the event as well as sending communications between the server and clients through network packets.

Custom Text Formatting - requires an understanding of regex expressions and matching bracket pairs in a string.

Tree generation - requires the use of generating random points within a 3D inequality to find branch locations, generating Bezier curves as branches, and finally placing merging metaballs for foliage.

Boats and potters - requires the development of new renderers.

There are many complex topics to be understood when developing a mod of this scale. Mathematics & statistics are frequently used, as well as advanced randomization, events, registries, object-oriented design, suppliers, threading, reflection, codecs, and JSON.

Card Game Repository

For software engineering class, my group and I were required to develop a webpage with a back-end server. My group and I decided to develop a card game repository where one could login, and host specific card games: blackjack, Exploding Kittens, Uno, and more. The server was to be programmed in Java with servlets and SQL interactions. The client web pages were built with HTML, CSS, and JavaScript.

This was a bit ambitious, and we certainly did not complete our original goal of 5+ playable games with bots and online matchmaking, however we made impressive progress. By the end of the semester, we had developed a server, capable of interacting with SQL to store game data on blackjack, Exploding Kittens, and Uno. The server had game logic completely implemented for 2-person blackjack and 2-6 player Uno with special rules. Unfortunately, Exploding Kittens was not able to be completed.

By the end of the project, a user could login to an account on the server via their browser and host a match of either blackjack or Uno. They would then receive a lobby code which they could send to their friends to join the game or host publicly for others to happen upon. The game would play out just as it normally would, preventing players from making illegal moves and keeping track of win / loss conditions. At the end of a match, or in the event of a disconnected player, all players would be returned back to the main menu where they could enjoy some music and view their game statistics.

Minecraft Block Image Converter

Minecraft block image converter is a program that I developed during software engineering. This program has the ability to convert standard images into images built with Minecraft blocks and place them in the world in a 2D or 3D format.

By default, a 2D format is used however one can change the program to output an image made for use with Minecraft maps which are generated from a top-down approach and individual map pixels are based on the color of the block the map is looking at, and it's elevation compared to its neighbors.

The program also has the ability to change the size of the outputted image, image filters, such as grey scale, monochrome, inversion, and others. Upon running the program, the user will be prompted to choose a color scaling, typically, the program compares colors using RGB however the user may select to use CIE-L*ab for a more accurate image.

The images displayed here have all been generated by the program. The fox is the output of the selections show in the GUI, RGB was used as well as the average color of each Minecraft block. The Chameleon is an example of a 3D image. The map must be generated through 3D lines of blocks.
